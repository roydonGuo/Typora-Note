# 上传已存在的本地文件

```sh
git init
git add .
git commit -m "描述"
git remote add origin 地址
git checkout -b main // 切换分支为main
git pull --rebase origin master //下拉查看是否有内容
git push -u origin master //推送到master分支
git status //查看状态，检查用到
```
